Browser Use Cloud API

Get live screenshots while the agent performs tasks in the browser

Overview

The Browser Use Cloud API allows you to run browser automation tasks and monitor them in real-time. This demo shows how to create a task and view a live preview of the browser as the agent performs the requested actions.

Implementation Guide

1 Create a new task using the API

First, send a POST request to create a new browser task:

import requests
import json

api_key = "your_api_key_here"
url = "https://api.browser-use.com/api/v1/run-task"

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

data = {
    "task": "Go to google.com and search for Browser Use"
}

response = requests.post(url, headers=headers, data=json.dumps(data))
task_data = response.json()

2 Extract the live URL from the response

The API response contains a live_url that you can use to watch the browser in real-time:

live_url = task_data.get('live_url')

3 Embed the live URL in your application

Use an iframe to display the live browser view:

<iframe src="{live_url}" width="800" height="600"></iframe>
